Constructor

Create a constructor, specifically used to create the Person object

The constructor is an ordinary function, and the creation method is no different from ordinary functions. The difference is that the constructor is used to capitalize the first letter

The difference between constructor and ordinary function is the difference in calling method

Ordinary functions are called directly, and if the constructor is called using the new keyword

Execution flow of the constructor

a. Create a new object immediately

b. Set the newly created object to this in the function

c. Execute the code in the function line by line

Return the newly created object as the return value

2     this.name = name;

3     this.age = age;

4     this.gender = gender;

5     this.sayName = function(){

6         alert(this.name);

7     };

8 }
