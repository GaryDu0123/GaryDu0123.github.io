HTML

Basic grammar overview

● HTML tags are keywords surrounded by angle brackets, such as <html>.

● HTML tags usually appear in pairs, such as <html> and </html>, which called double tags. The first tag in the tag pair is the start tag, and the second tag is the end tag.

● Some special tags must be single tags (rare cases), such as <br/>, which called single tags.

Title tag <h1>-<h6>

Paragraph and line break tags

In a web page, to display text in an orderly manner, these texts need to be displayed in segments. In HTML tags, the <p> tag is used to define paragraphs. It can divide the entire web page into several paragraphs.

Paragraph and line break tags

In HTML, the text in a paragraph will be arranged from left to right until the right end of the browser window, and then automatically wrap. If you want a certain text to be forced to wrap, you need to use the wrap tag<br/>.

<div> and <span> tags

<div> and <span> have no semantics, they are just a box to hold content.

The div tag occupies one line(Block-level element), the following text is displayed on the next line, and the span is used to lay out on one line(Inline element)

List label

Unordered list

● The <ul> tag represents an unordered list of items in the HTML page. Generally, the list items are presented in bullets, and the list items are defined by the <li> tag.

1 <ul>

2 <li>list items 1</li>

3 <li>list items 2</li>

4 <li>list items 3</li>

5 </ul>

1. The items in the unordered list have no order levels, and they are parallel.

2. Only <li><li> can be nested in <ul><u>. It is not allowed to directly enter other tags or text in the <ul></ul> tag.

3. The space between <li> and </li> is equivalent to a container, which can hold all elements.

Unordered lists have their own properties, but in actual use, they will be set using CSS

Ordered list

● In HTML tags, the <ol> tag is used to define an ordered list, the list sorting is displayed in numbers, and the <li> tag is used to define the list items.

● The basic properties are the same as Unordered list.

1 <ol>

2 <li> list items 1</li>

3 <li> list items 2</li>

4 <li> list items 3</li>

5 </ol>

Custom list

● Custom lists are often used to explain and describe terms or nouns. There are no bullets before the list items in the definition list.

● In HTML tags, the <dl> tag is used to define the description list (or definition list). This tag will be used together with <dt> (define item/name) and <dd> (describe each item/name).

1 <dl>

2     <dt>nouns 1</dt>

3     <dd> nouns 1 describe 1</dd>

4     <dd> nouns 1 describe2</dd>

5 </dl>

1. <d></dl> can only contain <dt> and <dd>.

2. There is no limit to the number of <dt> and <dd>, often one <dt> corresponds to multiple <dd>

Form

In HTML, a complete form usually consists of three parts: form controls (also called form elements), prompt information, and form fields. The purpose of the form is to collect user information.

● Form controls:

● Contains specific form function items, such as single-line text input box, password input box, check box, submit button, reset button, etc.

● Prompt information:

● A form usually needs to contain some descriptive text to prompt the user to fill in and operate.

● Form field:

● It is equivalent to a container, which is used to hold all the form controls and prompt information. It can define the URL address of the program used to process the form data and the method of submitting the data to the server. If the form fields are not defined, the data in the form cannot be transmitted to the background server.

<label> label

The <label> tag defines a label (label) for the input element.

The <label> tag is used to bind a form element. When click on the text in the <label> tag, the browser will automatically turn the focus (cursor) to or select the corresponding form element to increase the user experience.

1 <label for="sex">male</label>

2 <input type="radio" name="sex" id="sex"/>

<textarea> form element

Usage scenario: When the user enters a lot of content, we can't use the text box form. At this time, we can use the <textarea> tag. In form elements, the <textarea> tag is a control used to define multi-line text input.

Use the multi-line text input control to enter more text. This control is common in message boards and comments.
